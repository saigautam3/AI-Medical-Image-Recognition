Redesign my Flask-based AI Medical Image Recognition application into a modern, production-ready AI healthcare web application using React.js.

The application should have a premium, clean, futuristic UI similar to ChatGPT, Vercel AI, Perplexity AI, or modern healthcare SaaS platforms.

Tech Requirements

Frontend:
- React.js
- Vite
- Tailwind CSS
- Framer Motion
- React Icons
- Axios
- React Router
- React Dropzone
- Recharts (for future analytics)
- Lucide Icons

Backend:
- Keep existing Flask backend.
- Existing Gemini Vision API should remain unchanged.
- React should communicate with Flask via REST APIs.

Design Theme

- Modern AI Healthcare Platform
- White + Light Blue + Indigo gradients
- Glassmorphism cards
- Rounded corners
- Soft shadows
- Smooth animations
- Responsive design
- Mobile-first layout
- Premium SaaS appearance
